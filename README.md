# Math Challenge Game 🧮

A simple command-line math game built with Python that generates random arithmetic problems and measures how quickly the player solves them.

## Features

- Random math questions
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Tracks incorrect attempts
- Measures total completion time
- Interactive terminal gameplay

## Technologies Used

- Python
- Random Module
- Time Module

## How It Works

1. Press Enter to start the game.
2. Solve 10 randomly generated math problems.
3. Enter the correct answer.
4. If the answer is wrong, the question is asked again.
5. Complete all questions as quickly as possible.
6. The game displays your total completion time.

## Example Output

```text
Press enter to start!

Problem #1: 8 + 4 = 12
Problem #2: 10 * 5 = 50
Problem #3: 9 - 3 = 6

Nice work! You finished in 18.42 seconds!
```
